## pytorch数据读取三个常用类：
    - Dataset
    - DataLoader
    - DataLoaderIter

## torch.utils.data.Dataset
1. __getitem__() 根据索引读取数据
```python
def __getiem__(self, index):
    img_path = self.data[index]
    img = skimage.io.imread(img_path)

    return img
```
2. __len__() 返回整个数据集长度
```python
def __len__(self):
    return len(self.data)
```
